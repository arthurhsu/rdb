Types

All columns in a table schema must be associated with a given type. Valid types of a column are defined in ColumnType

    enum ColumnType {
      "arraybuffer",
      "boolean",
      "date",
      "number",
      "string",
      "object"
    };

arraybuffer and object types are not indexable, which means columns of these types could not be used to create indices and predicates (except the isNull() and isNotNull() predicates).
